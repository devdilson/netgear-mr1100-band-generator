Netgear Nighthawk MR1100 Band Generator

Notes

  • This guide is only known to work for AT&T in the United States. Please create a Github issue if this works (or doesn't work) in other markets.
  • Band 66 is an exclusive band and cannot be aggregated with other bands. The band selector for this is: AT!BAND=09, "LTE B66 (AWS-3)", 0, 0, 2
  • MR1100-1A1NAS (AT&T) Bands: 1, 2, 3, 4, 5, 7, 12, 20, 29, 30, 46, 66 (an early model which does not have B14)
  • MR1100-2A1NAS (AT&T) Bands: 1, 2, 3, 4, 5, 7, 12, 14, 20, 29, 30, 46, 66
